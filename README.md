\# Explainable Generative AI Dashboard for Real-Time Financial Forecasting and Decision Support



This project is my Master's thesis implementation.



\## Minimum Viable Thesis (MVT)



\- Single asset: S\&P 500 (daily data since 2020)

\- Generative model: Variational Autoencoder (VAE)

\- Explainability: SHAP

\- Contextual data: news sentiment only

\- Dashboard: Streamlit

\- Real-time: pseudo real-time (script-based/hourly refresh)



\## Folder Structure (high-level)



\- `01\_Documentation/` – thesis docs, notes

\- `02\_Data/` – raw, processed, and external data

\- `03\_Notebooks/` – EDA, model training, XAI analysis

\- `04\_Src/` – clean Python code (data pipeline, models, dashboard)

\- `05\_Results/` – metrics, plots, reports
