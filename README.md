# explainable-generative-ai-financial-forecasting

Master's thesis project: explainable generative AI dashboard for real-time financial forecasting and decision support.

## Project structure
- `01_Documentation` – thesis drafts and notes
- `02_Data` – raw and processed datasets (ignored by git)
- `03_Notebooks` – EDA and experiment notebooks
- `04_Src` – scripts for data preprocessing, models, and dashboard
- `05_Results` – saved plots, metrics, model checkpoints (ignored by git)
- `06_Presentations` – slides and demo materials
